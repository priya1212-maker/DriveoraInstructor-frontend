# Instructor App (React Native)

This project has been migrated from React JS web to a React Native mobile app using Expo.

## Run the app

1. Open terminal in `instructor-app`
2. Install dependencies:

```bash
npm install
```

3. Start Expo dev server:

```bash
npm run start
```

4. Run on device/emulator:

```bash
npm run android
# or
npm run ios
```

You can also scan the QR code in Expo Go.

## Converted screens

- Dashboard
- Bookings
- Profile
- Feedback
- Settings
- Map (opens native maps route)
